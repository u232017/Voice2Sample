# Technology & Music Lab (S101)

A small lab repository for exploring how **technology and music** intersect—through experiments, prototypes, notes, and short assignments.

## What this project is
This repo is a workspace for:
- **Learning-by-building**: small demos and mini-projects related to music + software/hardware.
- **Documenting experiments**: what was tried, what worked, what didn’t, and what to try next.
- **Collecting course/lab materials**: prompts, references, and deliverables (if applicable).

## What you’ll find here
Depending on the lab stage, the repository may contain:
- Source code for interactive music tools or audio experiments
- MIDI / audio / DSP explorations
- Write-ups, reports, and reflections
- Assets (audio clips, screenshots, diagrams)

> If you don’t see much yet, that’s normal—this README is meant to scale as the project grows.

## Getting started
1. **Clone the repo**
   ```bash

   cd technology-music-lab-s101
   ```

2. **Explore the files**
   - Start with the top-level folders and any notes or lab instructions.
   - If there’s code, look for a `requirements.txt`, `package.json`, or similar setup file.
