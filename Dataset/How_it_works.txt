1. Audio preparation
    1.1 Go to Dataset -> 1.Convert_audio_to_wav -> detect_audio_extensions.py change the input and then execute it.
    1.2 Go to Dataset -> 1.Convert_audio_to_wav -> wav_convertor.py and check if all the file extensions are abiable.
    Then change the input/ output path and execute.
    1.3 Revise all the audios have been well converted, check the sum up.
2. Metadata transformation
    2.1 Go to Dataset -> 2.Json_to_csv to convert the metadata.json given by the dataset, change the path before executing.
3. Creation of CSVs
    3.1 Go to Dataset -> 3.Clean_csv -> csv_filter.py first change the input and output paths, these code is to filter the csv of metadata to be ready to create the Database
    and csvs to train the models. It also checks if all the audio matches with the adios stored in the system.
    3.2 Go to Dataset -> 3.Clear_csv -> create_functional_csv.py to create 3 diferent csvs, one for training the deep learning model, one for regular machine learning model 
    and the last to use as database. Remember to change the paths.

    !!To proceed with theis last step it is required to use essentia, if you are using linux or ios you can use it with -- -- verisons of pyhton not higher.
    If you are in windows you will need to use a virtual enviorment, follow the next comands:
    wsl --install
